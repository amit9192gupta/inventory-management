# 🚀 Inventory & Order Management System

## 📌 Project Overview

This project is a backend-based Inventory and Order Management System built to understand real-world application of Java and Spring ecosystem. It manages products, orders, and users efficiently with proper backend architecture.

---

## 🛠️ Tech Stack

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA (Hibernate)
* MySQL
* REST APIs
* Maven

---

## 🏗️ Architecture

The project follows **MVC Architecture**:

* Controller → Handles HTTP requests
* Service → Business logic
* Repository → Database interaction

---

## ✨ Features

* Add / Update / Delete Products
* Place Orders
* Order Management
* Pagination support
* Exception Handling
* REST API design
* Database integration using JPA

---

## 🗄️ Database Design

* Product Table
* Order Table
* User Table

(Proper relationships maintained using JPA mappings)

---

## 🔥 Key Learnings

* Real-world usage of Spring Boot
* Understanding of REST APIs
* Database design and relationships
* Handling exceptions and validations
* Pagination implementation
